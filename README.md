# Face Attendance System

A real-time face detection attendance system using OpenCV.

## What it does
- Opens webcam and detects faces in real-time
- Draws green box around detected face
- Press 'S' to save face image
- Saves images with name and timestamp

## Technologies
Python | OpenCV | Haar Cascades | NumPy

## How to run
```bash
pip install opencv-python numpy
python attendance.py
```

## Key Concepts
- Haar Cascade Classifier for face detection
- Real-time video processing with OpenCV
- Automated image capture and storage
